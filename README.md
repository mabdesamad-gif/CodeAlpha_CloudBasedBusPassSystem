# 🚌 Cloud-Based Bus Pass System — CodeAlpha

A cloud-based bus pass management system built with Flask, SQLAlchemy, and Bootstrap.

## Features
- User registration and login
- Bus pass request and management
- Admin dashboard for pass approval/rejection
- QR code generation for each pass
- Automatic pass expiration
- Cloud deployment on Railway

## Tech Stack
- Python / Flask
- SQLAlchemy / SQLite
- Bootstrap 5
- APScheduler
- QRCode

## Installation
```bash
pip install -r requirements.txt
python create_admin.py
python app.py
```
